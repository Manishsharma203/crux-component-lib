Avatar

An avatar is an Internet user's representation of himself or herself in the form of a two-dimensional icon.

Examples

Avatars are available in two variants i.e. round and rectangular (avatar-round, avatar-rect). All avatars can be of three standard sizes avatar-lg, avatar-md, avatar-sm.

Letter based avatars can be used as well using class avatar-text and futher use them with different sizes as well i.e. avatar-text-lg, avatar-text-md, avatar-text-sm.

Avatars can be used in groups for specific use cases using class avatar-group on parent container with variations for large and small sizes as well i.e. avatar-group-lg and avatar-text-sm.
