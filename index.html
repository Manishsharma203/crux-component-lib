Avatar

An avatar is an Internet user's representation of himself or herself in the form of a two-dimensional icon.

Examples

Avatars are available in two variants i.e. round and rectangular (avatar-round, avatar-rect). All avatars can be of three standard sizes avatar-lg, avatar-md, avatar-sm.

Letter based avatars can be used as well using class avatar-text and futher use them with different sizes as well i.e. avatar-text-lg, avatar-text-md, avatar-text-sm.

Images

Images can be used as responsive images which will adjust according to the parent's width or round/circular images.

Examples

Responsive Images

Use class img-responsive to give the image responsiveness. In addition to that classes cover and contain can also be used for the object-fit property.
Circular Images

For perfectly circular images give the image square dimesions i.e. euqal widht and height. You can also use class cover in combination with parent elements dimensions to achieve the same layout.

Inputs

Input fields are used to create interactive controls for web-based forms in order to accept data from the user.

Inputs fields are represented as groups using class input-group. Text can be suffixed or prefixed using input-title.

For checkbox and radio inputs classes check-input and radio-input should be used for input fields. Similiary for displaying error or success state error-input and success-input should be used.
