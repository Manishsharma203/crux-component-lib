Images

Images can be used as responsive images which will adjust according to the parent's width or round/circular images.

Examples

Responsive Images

Use class img-responsive to give the image responsiveness. In addition to that classes cover and contain can also be used for the object-fit property.
Circular Images

For perfectly circular images give the image square dimesions i.e. euqal widht and height. You can also use class cover in combination with parent elements dimensions to achieve the same layout.

Grid Layout

The responsive Grid layout adapts to various screen sizes and orientation,

Parent container must have class grid-col to achieve the desired layout. The total width of layout is divide into six equal imaginary width spaces. Each column can take width of any number of spaces by using class col-'space'.

For devices/screens less than or equal to 600px view port width class col-'space'-sm should be used. And similiary class col-'space'-md can be used for less than or equal to 768px view port width screens.

    Width spaces are as follows:
  • col-1
  • col-2
  • col-3
  • col-4
  • col-5
  • col-6
